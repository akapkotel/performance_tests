
Welcome to Python Graphics Library Comparison's documentation!
==============================================================

This is a project to benchmark and compare different Python 2D graphics libraries.
The documentation and timings can be recreated by anyone.

Step 1
------

Make sure ``requirements.txt`` is loaded for your environment.

Step 2
------

Run ``src/__main__.py`` which should:

* Run all tests
* Generate all graphs using Matplotlib
* Generate all documents using Sphinx

Step 3
------

Look at the resulting documents in ``doc/build``
